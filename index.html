Lesson 2

In this lesson, we learned more about parameters and how they can be used in functions in order to generalize them, as well as the benefits of doing so.

What are the benefits of writing functions that use parameters and return? Try to list at least two.

Writing a function that uses a parameter and return is helpful because you don't have to write multiple functions that do the same thing, making your code repetitive in nature. Instead, using parameters and returns help keep your code less bulky allowing for easier comprehension as well as easier debugging.
